# Yazılım Yapımı Dersi Proje Ödevi-FİNAL

-------------------------------------------------------------------
## Scrum ile Geliştirilecek Yazılım Yapımı Projesi (Borsa Projesi)
-------------------------------------------------------------------

 Windows Form Uygulaması  

Geliştirilen uygulamada kullanıcılar sisteme üye olup giriş yapmaktadır. Sistemde admin ve kullanıcı girişleri bulunmaktadır. 
Kullanıcılar istediği üründen istediği miktarda alım-satım işlemini, sisteme para yatırma ve satın aldığı/sattığı ürünleri filtreleyip 
istenilen özelliklerdeki işlemleri Excel'e aktarma işlemlerini gerçekleştirebilmektedir.
Para yatırma işleminde istenilen para birimine uygun olarak para yükleyebilmektedir. Türk lirasından farklı bir değer girildiyse adminin para
talebini onayladığı zamanki kur değerlerine göre sisteme yüklenen para birimini hesaplamaktadır.
Sistem satın alınan ürünün %1'lik kısmını muhasebe işlemleri adı altında kesinti uygulamaktadır.
